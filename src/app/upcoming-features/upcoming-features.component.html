Personal Expense Tracker

This is an expense tracking application developed using Angular and NgRx for state management. Currently, it uses local storage.

Add Expense Entry

Allows the user to add new expense entries.

View Expense List

View all added expenses in a list format.

Edit Expense

Edit existing expense details easily.

Delete Expense

Remove unwanted expenses from the list.

Summary

View the total expense summary, including a category breakdown.

Next Updates

Add 'Expense From' Field

A new field to track which debit card was used for each expense. This will help users maintain a balance sheet for each card.

Backend Integration

The current app uses NgRx with local storage, but it is ready to be integrated with a backend for real-time data management.
